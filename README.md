# Task Assignment 04 – Object-Oriented Programming (OOP)

This repository contains my solution for **Task Assignment 04** as part of the **DEPI Scholarship Program**.
The task focuses on applying core Object-Oriented Programming (OOP) concepts using C#.

---

## 📌 Overview

The solution is divided into three main projects, each targeting specific OOP principles:

### 🔹 First Project – 3D Point Class

* Implemented a `Point3D` class with constructor chaining
* Overrode `ToString()`, `Equals()`, and operator overloading (`==`, `!=`)
* Implemented sorting using `IComparable`
* Added cloning functionality using `ICloneable`

---

### 🔹 Second Project – Maths Operations

* Created a static `Maths` class
* Implemented basic arithmetic operations:

  * Add
  * Subtract
  * Multiply
  * Divide (with exception handling)

---

### 🔹 Third Project – E-Commerce Discount System

* Designed a flexible discount system using:

  * **Abstraction**
  * **Inheritance**
  * **Polymorphism**
* Implemented multiple discount strategies:

  * Percentage Discount
  * Flat Discount
  * Buy One Get One (BOGO)
  * No Discount
* Created different user types:

  * Regular User
  * Premium User
  * Guest User
* Integrated user input to calculate final price dynamically

---
